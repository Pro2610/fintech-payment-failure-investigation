# fintech-payment-failure-investigation
# Why Transactions Fail - Fintech Payment Investigation

# Business Problem

A fintech platform is experiencing revenue loss due to failed transactions.

The goal of this investigation is to identify:
- where payment failures happen,
- what factors increase failure probability,
- how much revenue is lost,
- and what business actions could reduce failures.

 # Investigation Questions

 Key questions:

1. At which payment stage do failures happen most often?
2. Which payment methods have the highest failure rate?
3. Are some processors performing worse than others?
4. Which merchants generate the highest failed revenue?
5. What patterns indicate potential operational issues?

# Dataset

- 50,000 transactions
- 220 merchants
- Multiple payment methods (card, ach, wallet)
- Payment lifecycle stages:
  - authorization
  - capture
  - settlement
